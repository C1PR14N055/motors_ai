### Setup

#### Linux

```zsh
$ sudo apt install python3
$: pip3 install virtualenv
$: source venv/bin/activate
$: pip3 install -r requirements.txt
$: python3 run.py
```

#### OSX

```zsh
$: brew install python3
$: pip3 install virtualenv
$: source venv/bin/activate
$: pip3 install -r requirements.txt
$: python3 run.py
```
